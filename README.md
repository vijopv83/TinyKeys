# 🎮 TinyKeys — Learn with Pip 🦊

A delightful, interactive learning experience built entirely in a single HTML file.

TinyKeys combines gamification, animations, and intuitive UX to create an engaging environment where users can learn, play, and track progress—all without any frameworks or dependencies.

Built for developers, designers, educators, and product thinkers who want to explore how far pure HTML, CSS, and JavaScript can go.

## ✨ Features

- 🎮 **Gamified Learning** – Interactive exercises with instant feedback
- 🧠 **Multi-Mode Learning** – Flashcards, quizzes, and mini-games
- 👥 **Player Profiles** – Track progress, streaks, and XP
- 🏆 **Leaderboard System** – Compare performance across players
- 🎨 **Rich Animations** – Micro-interactions and transitions throughout
- 📊 **Progress Tracking** – Visual indicators for learning milestones
- ⚡ **Single File App** – Runs entirely from one HTML file
- 🌙 **Modern UI Design** – Clean, responsive, and engaging layout

---

## 🎯 What Makes This Special

- No frameworks. No build tools. No dependencies.
- Fully functional app in **one HTML file**
- Production-level UX patterns implemented from scratch
- Designed as both a **learning tool** and a **frontend showcase**
